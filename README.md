# ISWDP_Level1_Training
Participation in ISWDP Level 1 training covering semiconductor device fundamentals, TCAD-based design workflow, and basic device simulation.

##  Overview
Completed Level 1 training under the India Semiconductor Workforce Development Program (ISWDP), supported by Samsung and Synopsys.  
The program focused on semiconductor device fundamentals, technology CAD (TCAD), and basic device simulation and analysis.


##  Key Learnings
- Basics of semiconductor physics  
- Structure and working of semiconductor devices  
- Introduction to TCAD (Technology Computer-Aided Design)  
- Device simulation and analysis concepts  
- Semiconductor fabrication fundamentals  


##  Topics Covered
- PN Junction Diodes  
- MOSFET Basics  
- Carrier Transport Mechanisms  
- Device Characteristics and Analysis  
- 2D Device Creation using TCAD  


##  Tools & Technologies
- Synopsys TCAD Tools  
- Semiconductor Device Modeling  
- Simulation & Analysis Techniques  

##  Work Done
- Explored device simulation workflow  
- Understood basic 2D device creation  
- Analyzed electrical characteristics of devices  
- Studied semiconductor behavior through simulations  


## Skills Gained
- Semiconductor Device Understanding  
- TCAD Simulation Basics  
- Analytical Thinking  
- Device-Level Problem Solving
